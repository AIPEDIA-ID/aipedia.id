# Short-form Video Script

| Time | Visual/shot | Spoken line or on-screen text | Purpose |
|---|---|---|---|
| 0–3 sec | | | Hook |
| 3–10 sec | | | Problem/desire |
| 10–25 sec | | | Demo/proof |
| Final | | | Offer and CTA |

For 15 seconds, keep one claim and one proof. For 30–60 seconds, add an objection and a pattern interrupt every 5–10 seconds.
