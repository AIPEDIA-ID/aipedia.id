## Break-Even Analysis Template

```markdown
# Break-Even Analysis - [Nama Produk/Bisnis]

## Fixed Costs
- Rent: Rp [jumlah]/month
- Salaries: Rp [jumlah]/month
- Insurance: Rp [jumlah]/month
- Utilities (fixed): Rp [jumlah]/month
- Other Fixed Costs: Rp [jumlah]/month
**TOTAL FIXED COSTS: Rp [total]/month**

## Variable Costs (per Unit)
- Direct Materials: Rp [jumlah]/unit
- Direct Labor: Rp [jumlah]/unit
- Packaging: Rp [jumlah]/unit
- Shipping: Rp [jumlah]/unit
- Other Variable: Rp [jumlah]/unit
**TOTAL VARIABLE COST: Rp [total]/unit**

## Pricing
- Selling Price per Unit: Rp [jumlah]

## Calculations

### Contribution Margin per Unit
```
Contribution Margin = Selling Price - Variable Cost
                    = Rp [harga] - Rp [variable]
                    = Rp [hasil]/unit
```

### Contribution Margin Ratio
```
CM Ratio = Contribution Margin / Selling Price × 100%
         = Rp [hasil] / Rp [harga] × 100%
         = [%]
```

### BEP in Units
```
BEP Units = Fixed Costs / Contribution Margin
          = Rp [fixed] / Rp [CM]
          = [hasil] units/month
```

### BEP in Revenue
```
BEP Revenue = Fixed Costs / CM Ratio
            = Rp [fixed] / [ratio]
            = Rp [hasil]/month
```

## Profit Analysis

| Units Sold | Revenue | Variable Costs | Fixed Costs | Total Costs | Profit/Loss |
|------------|---------|----------------|-------------|-------------|-------------|
| 0 | Rp 0 | Rp 0 | Rp [fixed] | Rp [fixed] | (Rp [loss]) |
| BEP Units | Rp [hasil] | Rp [hasil] | Rp [fixed] | Rp [hasil] | Rp 0 |
| BEP + 100 | Rp [hasil] | Rp [hasil] | Rp [fixed] | Rp [hasil] | Rp [profit] |
| BEP + 500 | Rp [hasil] | Rp [hasil] | Rp [fixed] | Rp [hasil] | Rp [profit] |

## Safety Margin Analysis

| Metric | Value |
|--------|-------|
| Expected Monthly Sales (Units): | [jumlah] |
| BEP (Units): | [jumlah] |
| Margin of Safety (Units): | [hasil] |
| Margin of Safety (%): | [%] |

**Note: Safety Margin should be > 30% for healthy business**
```
