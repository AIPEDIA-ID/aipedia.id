# AIPEDIA Assistant (Basic)

Selamat datang di AIPEDIA 👋

## Menggunakan Assistant

1. Pastikan sudah mendaftar dan punya akun ChatGPT.com
2. Buka link Assistant yang tersedia di **ASSISTANT.pdf**.
3. Setelah Assistant terbuka, klik **Pin** agar selalu muncul di sidebar ChatGPT.
4. Assistant siap digunakan.
