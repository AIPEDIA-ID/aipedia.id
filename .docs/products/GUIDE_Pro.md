# AIPEDIA Assistant Pro

Selamat datang di AIPEDIA Pro 👋

Produk ini terdiri dari dua bagian:

- AIPEDIA Assistant
- AI Skills

---

# 1. Menggunakan Assistant

1. Pastikan sudah mendaftar dan punya akun ChatGPT.com
2. Buka link Assistant yang tersedia di **ASSISTANT.pdf**.
3. Klik **Pin** agar Assistant selalu muncul di sidebar.
4. Assistant siap digunakan.

---

# 2. Menggunakan AI Skills (Codex)

1. Download Codex `https://chatgpt.com/codex`
2. Extract **AI Skills Bundle**.
3. Pilih skill yang ingin digunakan, misalnya **Selo**, **Mita**, atau **Gita**.
4. Buka project Anda di Codex.
5. Tambahkan folder skill tersebut ke dalam project.
6. Ikuti petunjuk pada **README.md** di dalam setiap skill.
7. Mulai bekerja bersama Codex.

> Gunakan hanya skill yang diperlukan agar context tetap efisien.
