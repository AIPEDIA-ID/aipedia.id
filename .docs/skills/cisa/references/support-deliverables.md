# Support Deliverables

## Macro

`Empathy → confirm facts → resolution/next step → timeframe → escalation path → closing.`

Use placeholders: `[nama]`, `[nomor pesanan]`, `[tanggal]`, `[aksi]`, `[estimasi]`. Never promise an outcome outside policy.

## Escalation matrix

| Tier | Trigger | Owner | SLA | Customer update | Resolution authority |
|---|---|---|---|---|---|
| 1 | Standard request | CS | | | |
| 2 | Refund/late order | Lead | | | |
| 3 | Legal, safety, viral complaint | Manager | | | |

Track first response time, resolution time, CSAT, reopen rate, and escalation rate.
